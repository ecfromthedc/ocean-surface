Ocean browser extension context

Make the side panel visible to the agent.

The Ocean Chrome extension already runs the Leptos Surface app inside an MV3 side panel. The missing piece is explicit extension-origin and active-tab context on every agent turn.

Current state

wrapper works · context absent

What exists

The extension is an MV3 side-panel wrapper around the same Leptos/WASM Surface app used by the web PWA.

What is missing

The side panel can chat with Ocean, but does not yet send browser-tab context into /v1/agent/turns.

Implementation plan

5 steps
1

Extract extension-mode detection

Move the existing chrome-extension:// check into a reusable helper such as is_chrome_extension().

2

Send the correct client type

Use surface-extension from the side panel, otherwise keep surface-web.

3

Add extension permissions

Add tabs, activeTab, and scripting if URL/title/selection capture is required.

4

Bridge JS tab context into WASM

Expose window.__oceanExtensionContext(), query the active tab, and read selected text via chrome.scripting.executeScript.

5

Choose transport

Fast path: formatted guidance. Clean path: add typed client_context through ocean-core, daemon, and agent context.

JS bridge sketch

extension/ocean_extension_context.js
window.__oceanExtensionContext = async function () {
  const [tab] = await chrome.tabs.query({ active: true, currentWindow: true });

  let selection = "";
  try {
    const results = await chrome.scripting.executeScript({
      target: { tabId: tab.id },
      func: () => window.getSelection()?.toString() || "",
    });
    selection = results?.[0]?.result || "";
  } catch (_) {}

  return {
    surface: "chrome_extension_side_panel",
    active_tab: {
      id: tab?.id,
      url: tab?.url || "",
      title: tab?.title || "",
    },
    selection,
  };
};

Why it matters

agent awareness
“You’re in the Ocean Chrome extension side panel.” “The active tab is GitHub PR #123.” “You selected this error text; I’ll use it.” “I can use browser tools against this Chrome session.”

mostly ready
